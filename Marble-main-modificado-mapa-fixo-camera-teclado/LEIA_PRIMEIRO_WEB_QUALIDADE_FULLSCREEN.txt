Versão: qualidade Web + tela cheia

O que foi ajustado:
1. Viewport do projeto: 1920x1080.
2. Stretch mode: viewport.
3. Stretch aspect: expand.
4. Escala 3D: 1.0.
5. MSAA 3D: 2x.
6. HiDPI permitido.
7. Mantém velocidade 1x/2x/3x.
8. Mantém largada sem travar.

Como exportar:
1. Abra este projeto no Godot 4.4.
2. Vá em Projeto > Exportar.
3. Escolha Web.
4. Em Caminho de Exportação, exporte como index.html.
5. Thread Support deve ficar desativado.
6. Clique em Exportar Projeto.

Como testar:
1. Entre na pasta exportada.
2. Abra CMD nela.
3. Rode:
   py -m http.server 5500
4. Abra:
   http://localhost:5500

Tela cheia:
- Para tela cheia real no navegador, aperte F11.
- Dentro do site, use iframe/container com width:100vw e height:100vh.

Se ficar pesado:
- reduza para 80-100 bolinhas, ou
- tire MSAA 2x em Configurações do Projeto > Renderização > Anti Aliasing.
